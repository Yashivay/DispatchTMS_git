# Dispatch TMS - Bootstrap 5 Website

A professional Bootstrap 5 responsive website for Dispatch Transportation Management System.

## Project Structure

```
DispatchTMS_git/
├── index.html           # Main home page
├── css/
│   └── styles.css       # Custom CSS styling
├── js/
│   └── script.js        # Interactive JavaScript features
└── README.md            # Project documentation
```

## Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Bootstrap 5** - Built with the latest Bootstrap framework
- **Modern Navigation** - Sticky navbar with smooth scrolling
- **Hero Section** - Eye-catching landing banner
- **Features Section** - Showcase key features
- **Services Section** - Highlight available services
- **Testimonials** - Customer feedback and reviews
- **Call-to-Action** - Conversion-focused contact section
- **Professional Footer** - Complete contact and social information

## Sections Included

1. **Navigation Bar** - Responsive navbar with menu links
2. **Hero Section** - Full-screen welcome banner
3. **Features** - 3-column feature showcase
4. **Services** - Service offerings
5. **How It Works** - Process explanation
6. **Testimonials** - Customer reviews
7. **Call-to-Action** - Sign up/contact section
8. **Footer** - Copyright and company info

## Technologies Used

- **HTML5** - Semantic markup
- **Bootstrap 5.3.0** - CSS Framework
- **CSS3** - Custom styling
- **JavaScript** - Interactive features
- **Responsive Design** - Mobile-first approach

## Getting Started

1. Clone the repository
2. Open `index.html` in your web browser
3. Customize content in `index.html`
4. Modify styles in `css/styles.css`
5. Add interactivity with `js/script.js`
